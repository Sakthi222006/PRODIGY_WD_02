# Stopwatch Web Application ⏱️

This project is a responsive Stopwatch Web Application developed as part of the Prodigy Infotech Web Development Internship.

The application allows users to:
- Start the stopwatch
- Pause the stopwatch
- Reset the stopwatch
- Record lap times

The project is built using HTML, CSS, and JavaScript with a clean and user-friendly interface.

---

## 🚀 Features

- Start Stopwatch
- Pause Stopwatch
- Reset Stopwatch
- Lap Time Tracking
- Responsive Design
- Interactive UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
